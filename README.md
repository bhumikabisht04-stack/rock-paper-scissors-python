# Rock Paper Scissors 🎮

This is my second Python project, built while learning the basics of programming.

The game lets the user choose Rock, Paper, or Scissors, while the computer makes a random choice. It then compares both choices and displays whether the player wins, loses, or if it's a draw.

## Features

- User vs Computer gameplay
- Random computer choice
- Win, lose, and draw detection
- Simple and beginner-friendly code

## Technologies Used

- Python

## What I Learned

While building this project, I practiced:
- Taking user input
- Using the `random` module
- Writing `if`, `elif`, and `else` statements
- Combining different concepts to create a complete program

I'm currently learning Python and building small projects to improve my problem-solving and programming skills. More projects will be added as I continue learning.
